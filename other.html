LiteFlux LF2 (PDF manual)
The LiteFlux LF2 was a AAA LED flashlight with an incredibly versatile, incredibly terrible interface. The manual had multiple flowcharts showing you how to program the unit in a form of binary, with "head unscrewed one quarter turn" as a 0 and "head unscrewed one half turn" as a 1. The following files are, to my knowledge, the only surviving copy of this documentation available anywhere online.
  • PDF user manual
  • InDesign CS5 source for user manual
  • Operation mode exchange flowchart | gliffy source | PNG
  • Function access flowchart | gliffy source | PNG
  • Programming function flowchart | gliffy source | PNG
